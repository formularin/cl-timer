cl-timer: A cubing timer for the terminal

When I got tired of opening a web browser every time I wanted to cube and fearing losing my scrambles, I decided to create cl-timer. cl-timer is inspired by vim, having a command system that allows the functionality of most timers. This includes the ability to:

  • Change scramble type and length
  • See all averages and scrambles
  • Create and move between sessions
  • delete, dnf, and plus-two times

Right now it's only available for macOS
