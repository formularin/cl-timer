CL Timer Documentation

Startup Command: cl-timer. As of right now, there are no command-line options.

Once you are in a session, press ":" to enter command mode. To exit command mode, press the escape key.

Commands

s - change the sessions's settings

Syntax: s (sl | p) <value>

  • sl - scramble length. Accepts any integer value.
  • p - puzzle (for the scramble). Accepts any integer value between 2 and 7 (inclusive).

Example Usage: s p 7 - set the puzzle to 7x7

i - show solve data

Syntax: i [<index>]

  • No arg - Opens session file in read-only mode, showing scrambles and averages for all times.
  • index - Must be an integer between zero and the number of solves in the session. Shows data for solve with that index.

Example Usage: i 13 - show data for the 13th solve

c - change session

Syntax: c <session-name>

  • session-name - Any valid file name. Opens a session with that name. If no such session exists, it creates a new one.

Example Usage: s comp-practice - open session called comp-practice

rm - delete solve

Syntax: rm (all | <index>)

  • all - Deletes all times in session
  • index - index - Must be an integer between zero and the number of solves in the session. Deletes solve with that index.

Example Usage: rm 13 - delete the 13th solve

d - mark most recent solve as DNF

p - mark most recent solve as plus-two

q - quit the program
