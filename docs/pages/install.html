Install

As of right now, these installation options only work on macOS

For all of these options, you will have to have python installed. To do this, go to python.org and download the latest version of python.

Install by Downloading Source Code

  1. Download this file
  2. Double-click the file icon. A new folder should be created
  3. Open the folder, select all the files, and press cmd + c to copy them
  4. Open a new Finder window, and go to the outermost folder (should be in "locations")
  5. Click on "Macintosh HD"
  6. Press cmd + shift + . to show hidden files
  7. Go to usr, then local, then bin
  8. Press cmd + v to paste the files you copied before

Install Using pip

  1. Press cmd + spacebar, type "terminal", and press enter
  2. In the window that pops up, type "pip install cl-timer", then press enter

After installing using one of the above methods, whenever you open terminal and enter the command "cl-timer", the program will start.
