Install

As of right now, these installation options only work on macOS

For all of these options, you will have to have python installed. To do this, go to python.org and download the latest version of python.

Install Using pip

  1. Press cmd + spacebar, type "terminal", and press enter
  2. In the window that pops up, type "pip install cl-timer", then press enter

Install by Downloading Source Code (Not recommended)

This method is very complicated, and it is much easier to install with pip.

  1. Download this file and unzip it.
  2. Use the 'mv' command to move the folder into the site-packages directory in your python interpreter. Rename the folder to cl-timer if you have to. For me that is '/Library/Frameworks/Python.framework/Versions/3.8/lib/python3.8/site-packages/'. You may need to use 'sudo'.
  3. Download this file.
  4. Move this file to the bin of you python interpreter. For me, this is '/Library/Frameworks/Python.framework/Versions/3.8/bin/'.
  5. Run 'chmod +x <path-to-cl-timer-executable>'

After installing using one of the above methods, whenever you open terminal and enter the command "cl-timer", the program will start.

IMPORTANT

CL Timer has the ability to detect a spacebar held for a long time, then released as a single keypress. It does this by considering key repeat speed, and in order for it to work your delay until key repeat has to be as low as possible.


To shorten your delay until key repeat:

  1. Open the System Preference app
  2. Go to the Keyboard settings
  3. Move the slider below "delay until repeat" as far to the right (the side labeled "short") as possible
